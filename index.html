Servicios

¿Por qué tener una pagina web?

La implementación de una plataforma web para tu negocio ayudará a que muchos procesos internos y externos sean optimizados. Tales como la comunicación, reducción de costos, mayor productividad y brindar una experiencia única al usuario.

La creación de una página web te permite transmitir una correcta información sobre los productos o servicios de tu marca basada en la experiencia de usuario (UX).

¿Cuál es mi metodología de trabajo?

Planificar y desarrollar sitios webs que respondan tanto a las necesidades de la empresa como a las de los usuarios. Manteniendo la siguiente gestión:

HTML5

CSS3

Buen nivel para estructurar diseños. Maquetación de sistemas unidimensional (flexbox) y bidimensional (grid). Correcto uso de la metodología BEM, además de Responsive Design y Mobile First.

JavaScript

Buen nivel con vanilla JS. Me brinda las herramientas necesarias como lenguaje de programación para definir la lógica de todos los proyectos, ya que puede dotar de interactividad y dinamismo.

PWA

Utilizo la tecnología de Progressive Web App para que la base de las páginas webs tengan una estética y funcionamiento que se asemejen enormemente a una App nativa. Además me brinda excelentes características, ya que es segura, rápida, responsive y offline.

Netlify

Buen nivel con la tecnología Netlify. Utilizo este servicio para hospedar mis repositorios de código abierto ya que me brinda hosting gratis y seguridad. También me permite realizar cualquier mantenimiento o cambio en línea de una forma rápida y sencilla acompañado de GitHub.

Firebase

Nivel básico en Firebase. Utilizo esta tecnología para la creación de tanto aplicaciones webs como móviles y su desarrollo. Además me brinda la excelente herramienta de apoyo del lado del servidor (Backend) de la cual carezco de conocimiento actualmente.

Cronómetro - JS

Proyecto de práctica en JavaScript de una temática sobre el funcionamiento del Cronómetro. La practica esta enfocada en hacer funcionar los botones y que corra el tiempo, a su vez una animación de bolilla que gira 360º grados completando un minuto. Se usaron los conocimientos de: ObjetoMath, Condicionales, Objetos, Funciones Flechas, Variables y el DOM.

Licuadora - JS

Proyecto de práctica en JavaScript de una temática sobre el funcionamiento de una Licuadora. La practica esta enfocada en hacer funcionar los botones de encendido y apagado de la Licuadora. Se usaron los conocimientos de: Condicionales, Funciones, Variables.

Formulario
