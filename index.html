Sobre Mí

¡Hola! Mi nombre es Alfred Barragán. Soy estudiante autónomo en Desarrollo Web. Creo páginas con diseños minimalistas, modernas y profesionales. Además, desarrollo interfaces de usuario para aplicaciones móviles Android.

HTML5

Buen dominio con la sintaxis para lograr un código flexible, limpio y profesional. Correcto uso de HTML semántico para brindar una mejor optimización y ventaja respecto al posicionamiento SEO.

CSS3

Buen nivel para estructurar diseños. Maquetación de sistemas unidimensional (flexbox) y bidimensional (grid). Correcto uso de la metodología BEM, además de Responsive Design y Mobile First.

JavaScript

Buen nivel con vanilla JS. Me brinda las herramientas necesarias como lenguaje de programación para definir la lógica de todos los proyectos, ya que puede dotar de interactividad y dinamismo.

PWA

Utilizo la tecnología de Progressive Web App para que la base de las páginas webs tengan una estética y funcionamiento que se asemejen enormemente a una App nativa. Además me brinda excelentes características, ya que es rápida, responsive y offline.

Netlify

Buen nivel con la tecnología Netlify. Utilizo este servicio para hospedar mis repositorios de código abierto ya que me brinda hosting gratis y seguridad. También me permite realizar cualquier mantenimiento o cambio en línea de una forma rápida y sencilla acompañado de GitHub.

React JS

Nivel básico con la librería React JS. Me permite crear páginas óptimas y de buen rendimiento debido sus características. Además, utilizo React native expo que es un marco nativo para crear interfases de aplicaciones móviles tanto para la plataforma Android como iOS
